Além dos arquivos fonte em VHDL, por problemas na confecção do
testbench, foi feito um arquivo Waveform.vwf, o qual possui
uma simulação, também utilizando o modelsim, do funcionamento do
projeto. As imagens a seguir mostram a síntese do projeto, e as 
instruções para rodar o Waveform.

    - Sintese.jpg :: Imagem da síntese com os parâmetros pós-compilação.

    - Instrucao1.jpg :: Após adicionar o arquivo .vwf ao projeto, junto a todos
    os outros componentes, clique nele duas vezes, e este irá abrir uma nova janela.

    - Instrucao2.jpg :: Após abrir a nova janela, clique em Simulation -> Options.

    - Instrucao3.jpg :: Certifique-se que a opção para simulação está em ModelSim.

    - Instrucao4.jpg :: Clique no botão o qual a seta aponta para começar a simulação.

    - Funcionamento.jpg :: A imagem a seguir mostra a simulação, no qual pode-se
    observar que a mesma é completamente funcional.
