Sumário dos arquivos presentes no trabalho::


Além dos arquivos fonte e testbench em VHDL, o Projeto possui imagens, as quais:

 - trabSINTESE.jpg :: consiste na sintese dos dados apresentados pelo Quartus.

 - trabAND.jpg :: consiste no teste da operação AND, tirada do modelsim.

 - trabOR.jpg :: consiste no teste da operação OR, tirada do modelsim.

 - trabNOR.jpg :: consiste no teste da operação NOR, tirada do modelsim.

 - trabXOR.jpg :: consiste no teste da operação XOR, tirada do modelsim.

 - trabADD.jpg :: consiste no teste da operação ADD, com operação sem overflow.

 - trabADD_OV.jpg :: consiste no teste da operação ADD, com operação com overflow.

 - trabADDU.jpg :: consiste no teste da operação ADDU, com operação sem overflow.

 - trabADDU_OV.jpg :: consiste no teste da operação ADD, com operação com overflow, para mostrar que a operação não aciona a flag ovfl.

 - trabSUB.jpg :: consiste no teste da operação SUB, com operação sem overflow.

 - trabSUB_OV.jpg :: consiste no teste da operação SUB, com operação com overflow.

 - trabSUBU.jpg :: consiste no teste da operação SUBU, mostrando o funcionamento da flag zero.

 - trabSLT0.jpg :: consiste no teste da operação SLT, com a maior que b.

 - trabSLT1.jpg :: consiste no teste da operação SLT, com a menor que b.

 - trabSLTU0.jpg :: consiste no teste da operação SLTU, com a(unsigned) maior que b(unsigned).

 - trabSLTU1.jpg :: consiste no teste da operação SLTU, com a(unsigned) menor que b(unsigned).

 - trabSLL.jpg :: consiste no teste da operação SLL, com a shiftando b bits.

 - trabSRL.jpg :: consiste no teste da operação SRL, com a shiftando b bits.

 - trabSRA.jpg :: consiste no teste da operação SRA, com a shiftando b bits.

 - trabRTR.jpg :: consiste no teste da operação RTR, com a rotacionando b bits.

 - trabRTL.jpg :: consiste no teste da operação RTL, com a rotacionando b bits.
